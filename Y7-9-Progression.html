Year 7 Computer Science Theory of Knowledge

  • Understand the basics of computer networks and how the Internet works;
  • Understand computers are made of different components, including hardware and software;
  • Be able to identify basics computer hardware components;
  • Understand computers represent numbers and images in binary;
  • Understand computers need precise instructions to work;

Year 7 Computer Science: Skills

  • Can create multiple linked webpages using HTML with styling using CSS and multi-media contents such as images and videos;
  • Be able to identify selection and iteration in a given programming code;
  • Can devise algorithms in logical and precise sequence of instructions to solve simple problems using structured English or pseudocode;
  • Can confidently use at least three data types: integer, string and Boolean
  • Can program solutions to problems using variables, arrays, loops, and if statements;

Year 8 Computer Science: Theory of Knowledge

  • Understand the basics of computer networks and how the Internet works;
  • Understand computers are made of different components, including hardware and software;
  • Understand basic computer hardware components and how they work together;
  • Understand the factors affecting performance of a computer;
  • Understand computers represent text and image in binary;
  • Understand computers need precise instructions to work;
  • Understand the four principles of computational thinking: decomposition, pattern recognition, abstraction,and algorithm design;

Year 8 Computer Science: Skills

  • Can create multiple linked webpages using HTML with styling using CSS, both inline and external, and multi-media content;
  • Be able to identify basic computer hardware components;
  • Can perform binary to decimal conversion and vice versa on positive integers;
  • Can break down a large problem into smaller problems to reach a solution;
  • Can devise algorithms in logical and precise sequence of instructions to solve problems using structured English or pseudocode;
  • Understand the concept of data structure and be able to use one dimensional array in problem solving;
  • Can confidently use at least four data types: integer, string, float and Boolean;
  • Understand the concept of modular programming and be able to define and call functions with or without parameters;
  • Can program in Python and debug solutions to problems using variables, arrays, loops, if statements and functions;
